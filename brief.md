# Техническое задание: Лендинг-портфолио «Цифровой мастер»

> Главный документ проекта. Основан на research.md (апрель 2026).

---

## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта
Одностраничный лендинг-портфолио для получения заявок от потенциальных клиентов. Единственное целевое действие — **написать в Telegram или заполнить форму**.

### Целевая аудитория

| Сегмент | Кто | Задача |
|---|---|---|
| B2B малый бизнес | Владельцы компаний, предприниматели | Сайт, автоматизация, чатбот |
| B2C эксперты | Коучи, консультанты, фрилансеры | Лендинг, личный сайт |
| Стартапы | Основатели, продакты | AI MVP, веб-приложение |
| Digital-команды | Маркетологи, e-commerce | Автоматизация, ИИ-инструменты |

### Главный страх клиента
«Возьмёт деньги и пропадёт» — нужно закрыть через кейсы, процесс работы, прозрачные условия.

### 00. NAV — Навигация

**Состав:**
- Слева: логотип или имя (оранжевый акцент)
- По центру/справа: якорные ссылки → Услуги / Проекты / Обо мне / Контакты
- Крайний правый: кнопка «Написать» (оранжевая, малая)

**Поведение:**
- Sticky (прилипает при скролле)
- При скролле вниз: фон становится `rgba(10,10,15,0.95)` + `backdrop-filter: blur(12px)` (glass-эффект)
- На мобильном: гамбургер-меню

### 04. SERVICES — Услуги

**Заголовок:**
> Что я делаю

**Подзаголовок:**
> Три направления — одна цель: чтобы ваш бизнес работал лучше.

**4 карточки услуг:**

**1. Сайты и лендинги**
Иконка: 🖥 (или SVG-иконка монитора)
Описание: Сайт, который приводит клиентов, а не просто существует. Лендинги, портфолио, корпоративные сайты — под вашу задачу.
Результат: *Готово за 7–14 дней*

**2. Веб-приложения**
Иконка: ⚙️ (шестерня / интерфейс)
Описание: Личные кабинеты, CRM-инструменты, онлайн-сервисы. Когда нужно больше, чем просто страница.
Результат: *MVP за 2–4 недели*

**3. ИИ-агенты и чат-боты**
Иконка: 🤖 (робот / нейросеть)
Описание: Бот, который отвечает клиентам 24/7. Агент, который обрабатывает заявки без менеджера. Автоматизация рутины.
Результат: *Экономит 3–5 часов в день*

**4. Автоматизация процессов**
Иконка: ⚡ (молния / поток)
Описание: Связываю сервисы между собой, убираю ручную работу. Telegram, CRM, таблицы — всё работает само.
Результат: *Настройка за 3–7 дней*

**Стиль карточек:** Glassmorphism — `rgba(255,255,255,0.04)` фон, тонкая рамка `rgba(255,255,255,0.08)`, blur

### 05. PROCESS — Как мы работаем

**Заголовок:**
> Как проходит работа

**Подзаголовок:**
> Никаких сюрпризов. Только чёткий процесс и понятный результат.

**5 шагов (горизонтальный таймлайн на десктопе, вертикальный на мобильном):**

1. **Созвон / Заявка**
   Рассказываете задачу. Я задаю вопросы, предлагаю решение. Без лишних слов.

2. **Бриф и смета**
   Фиксируем объём, сроки и стоимость. Всё в документе — ничего на словах.

3. **Разработка**
   Делаю, показываю промежуточные результаты. Вы всегда знаете что происходит.

4. **Правки и сдача**
   Финальные корректировки входят в стоимость. Сдаю только когда довольны.

5. **Поддержка**
   Не пропадаю после сдачи. Помогу если что-то пойдёт не так.

### 10. FAQ — Частые вопросы

**Заголовок:**
> Частые вопросы

**7 вопросов (аккордеон):**

**Сколько стоит?**
Зависит от задачи. Лендинг — от 15 000 руб., Telegram-бот — от 10 000 руб., веб-приложение — от 30 000 руб. Точную стоимость назову после разговора.

**Как долго делается?**
Лендинг — 7–14 дней. Бот или автоматизация — 3–7 дней. Веб-приложение — от 2 недель. Всегда договариваемся о сроке до старта.

**Как проходит оплата?**
50% предоплата, 50% после сдачи. Для крупных проектов делим на 3 этапа.

**Сколько правок входит в стоимость?**
Неограниченные правки в рамках согласованного ТЗ. Если появляется новая задача — обсуждаем отдельно.

**Вы работаете по договору?**
Да, заключаем договор. Можно с ИП или физическим лицом.

**Что будет после сдачи проекта?**
Я остаюсь на связи. Первый месяц — бесплатная поддержка по вопросам использования.

**Вы работаете с небольшими бюджетами?**
Да. Начинаем с малого — потом расширяем. Лучше запустить простое и рабочее, чем долго делать идеальное.

## 4. ДИЗАЙН-СИСТЕМА

### Цвета

| Переменная | Цвет | Использование |
|---|---|---|
| `--bg-main` | `#0a0a0f` | Основной фон |
| `--bg-card` | `#12121f` | Фон карточек |
| `--bg-section` | `#0f0f1a` | Чередующийся фон секций |
| `--text-primary` | `#ffffff` | Заголовки |
| `--text-secondary` | `#a0a0b0` | Описания, подзаголовки |
| `--text-muted` | `#606070` | Плейсхолдеры, мелкий текст |
| `--accent` | `#ff8c00` | Кнопки, ссылки, акценты |
| `--accent-hover` | `#e67e00` | Hover-состояние кнопок |
| `--accent-glow` | `rgba(255,140,0,0.15)` | Свечение, aurora-пятна |
| `--border` | `rgba(255,255,255,0.08)` | Рамки карточек |

### Шрифты

**Основной шрифт:** `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
*(системный — быстро грузится, хорошо читается)*

**Моноширинный (для тегов, стека, typing-эффекта):** `'JetBrains Mono', 'Fira Code', monospace`

**Типографическая шкала:**
```
H1: clamp(40px, 7vw, 96px) | weight 800
H2: clamp(28px, 4vw, 52px) | weight 700
H3: clamp(20px, 2.5vw, 28px) | weight 600
Body: 16–18px | weight 400
Small: 14px | weight 400
```

### Настроение / Tone of voice

- Тёмный, технологичный, но тёплый (оранжевый смягчает холодность)
- Профессиональный, но не корпоративный
- Уверенный, но не кричащий
- Текст: простой, конкретный, на «вы»

## 5. ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ

### Кнопки

| Тип | Стиль | Использование |
|---|---|---|
| Primary | Оранжевый фон, белый текст, border-radius 8px | Главные CTA |
| Secondary | Прозрачный фон, оранжевая рамка, оранжевый текст | Вторичные действия |
| Ghost | Без фона, серый текст | Навигация, мелкие ссылки |

**Hover на всех кнопках:** подъём `translateY(-2px)` + усиление тени

### Формы

- Поля: тёмный фон `#1a1a2e`, тонкая рамка
- При фокусе: рамка становится оранжевой (`box-shadow: 0 0 0 2px var(--accent)`)
- Placeholder: `--text-muted`
- Кнопка отправки: Primary

### Анимации (обязательные)

| Анимация | Где | Техника |
|---|---|---|
| Typing-эффект | Hero, под заголовком | JS setInterval |
| Reveal при скролле | Все секции и карточки | Intersection Observer |
| Stagger на карточках | Услуги, портфолио | CSS `animation-delay` |
| Hover на карточках | Услуги, портфолио | `transform: translateY(-4px)` + тень |
| Aurora-фон | Hero | CSS `@keyframes` + radial-gradient |
| Glass-хедер при скролле | NAV | JS scroll event + CSS transition |
| Счётчик цифр | PROOF (статистика) | Intersection Observer + JS |
| Аккордеон | FAQ | CSS max-height transition |

### Адаптивность

| Брейкпоинт | Ширина | Особенности |
|---|---|---|
| Mobile | < 768px | Одна колонка, гамбургер-меню, крупные кнопки ≥44px |
| Tablet | 768–1024px | 2 колонки для карточек |
| Desktop | > 1024px | Полная сетка, hover-эффекты |

## 6. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

- **Один HTML-файл** — без фреймворков и внешних библиотек
- **Чистый HTML/CSS/JS** — нативные API браузера
- **Форма:** отправка через Telegram webhook (основной) + Formspree (fallback)
- **Шрифты:** системный стек (без Google Fonts для скорости)
- **Иконки:** SVG inline или Unicode-символы (без icon-библиотек)
- **Изображения:** WebP, lazy loading (`loading="lazy"`)
- **Performance:** Google PageSpeed ≥ 90
- **SEO:** meta title, description, OG-теги
