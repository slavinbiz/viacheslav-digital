## Чеклист тестирования

### Навигация
- [ ] Логотип «Цифровой.Мастер» → прокрутка к `#hero`
- [ ] Пункты меню (Услуги / Проекты / Процесс / Обо мне) → плавная прокрутка к секции
- [ ] Кнопка «Написать» → прокрутка к `#contact`
- [ ] Бургер-меню на мобильном открывается и закрывается
- [ ] Навбар закрепляется при скролле (появляется прогресс-бар)

### Hero
- [ ] Анимированный текст в строке `// Разрабатываю...` меняется циклически
- [ ] Кнопка «Обсудить проект» → прокрутка к `#contact`
- [ ] Кнопка «Посмотреть работы» → прокрутка к `#portfolio`
- [ ] Бейдж «Доступен для новых проектов» виден

### Счётчики (`#proof`)
- [ ] При появлении в viewport счётчики анимируются от 0 до значений: Проектов 14+, Лет опыта 3+, Ответ 2ч

### Портфолио (`#portfolio`)
- [ ] 6 карточек отображаются
- [ ] Первые 2 карточки — реальные скриншоты (`case-site.jpg`, `case-tma.jpg`)
- [ ] При наведении на первую карточку появляется кнопка «Открыть сайт →» → `https://viacheslav-digital.vercel.app`
- [ ] При наведении на вторую карточку — «Открыть в Telegram →» → `https://t.me/Ai_masterska_bot`

### FAQ (`#faq`)
- [ ] Тап на вопрос открывает/закрывает ответ (аккордеон)
- [ ] Одновременно открыт только один ответ

### Форма заявки (`#contact`)
- [ ] Поля: Имя, Email, Сообщение
- [ ] При отправке пустой формы — ошибка «Пожалуйста, заполните все поля.»
- [ ] При успешной отправке — форма скрывается, появляется экран успеха
- [ ] Кнопка «Отправить ещё» возвращает форму
- [ ] Кнопка отправки блокируется на время запроса (текст «Отправка...»)

### Отправка заявки
При отправке сообщение уходит в Telegram:

```

Chat ID:   888224075
```

Формат сообщения:
```
🆕 Новая заявка с сайта

👤 Имя: [имя]
📧 Email: [email]
💬 Задача: [сообщение]
```

### Адаптивность
- [ ] Десктоп (1200px+): меню горизонтальное, секции в 2-3 колонки
- [ ] Планшет (768px): меню сжимается
- [ ] Мобильный (до 480px): бургер-меню, одна колонка, кнопки на всю ширину

### Анимации
- [ ] Элементы появляются при скролле (класс `reveal`)
- [ ] Счётчики анимируются при первом попадании в viewport (срабатывают один раз)

## Частые ошибки и решения

| Ошибка | Решение |
|---|---|
| Заявка не приходит в Telegram | Проверить `TG_TOKEN` и `TG_CHAT_ID` в `index.html` (строки 1687–1688) |
| Скриншоты в портфолио не отображаются | Убедиться что `case-site.jpg` и `case-tma.jpg` лежат в корне проекта |
| Счётчики не анимируются | Проверить что `IntersectionObserver` поддерживается браузером |
| Меню не закрывается после клика | Проверить функцию `closeMobileMenu()` в JS |
