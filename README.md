# My Project

Кратко: стартовая заготовка проекта. Заполните `PLAN.md` информацией о стеке и задачах.

Как начать (пример):

1. Установите зависимости (если используете npm или pip).
2. Запустите локальный сервер/скрипт сборки.

Примеры команд (замените под стек проекта):

```
npm install
npm run dev
```

или

```
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Описание структуры и дальнейшие шаги — в `PLAN.md`.
