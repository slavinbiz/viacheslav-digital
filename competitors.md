# Анализ конкурентов для AI-специалиста (РФ)

## Цель
Создать продающий сайт для специалиста по ИИ с позиционированием в РФ (русский язык) и возможностью расширяться.

## Краткий обзор 5 примеров

### 1. Netpeak
- Услуги:
  - Стратегия
  - Performance Marketing
  - SEO
  - CRO
  - Аналитика и AI-автоматизация
- Позиционирование: результат (ROI), опыт 16+ лет
- Страницы: услуги, кейсы, отзывы, процесс, блог, FAQ
- Результат: метрики кейсов, отзывы, портфолио брендов

### 2. WebbyLab
- Услуги:
  - Full-cycle development
  - AI
  - Blockchain / IoT
  - UX/UI
  - DevOps
- Позиционирование: enterprise-сервисы, долгосрочные продукты
- Страницы: услуги, кейсы, команда, блог
- Результат: case studies, метрики, цитаты, retention 88%

### 3. Tilda
- Услуги:
  - конструктор сайтов
  - eCommerce
  - CRM
  - email-маркетинг
  - AI-генерация контента
- Позиционирование: no-code, быстро, дизайнерский продукт
- Страницы: продукты, шаблоны, галерея, кейсы, обучение
- Результат: галерея «Made on Tilda», живые примеры

### 4. SendPulse
- Услуги:
  - email, sms, viber
  - чат-боты
  - лендинги
  - автоматизация
  - веб-пуш
- Позиционирование: единственная платформа для маркетинга
- Страницы: продукты, цены, кейсы, academy
- Результат: цифры (3M пользователей, 4000 отзывов), шаблоны кейсов

### 5. IT-GRAD
- Услуги:
  - облака (private/public/hybrid)
  - colocation
  - DRaaS
  - CDN, WAF
- Позиционирование: надежность, российская инфраструктура
- Страницы: услуги, преимущества, партнёры, блог
- Результат: партнёры, SLA, compliance, кейсы

## Выводы для вашего сайта
1. Структурируй услуги по категориям (сайт, чат-боты, AI-агенты, внедрение, фотосессии)
2. Формулируй USP (одно предложение)
3. Обязательно: кейсы/результаты с метриками и отзывами
4. Сделай раздел `Портфолио` и `Цены`
5. Добавь страницу контактов + форму обратной связи

---

## Рекомендации по MVP
- Лэндинг + Услуги + Портфолио + Отзывы + Контакты
- Кейсы с результатами (до/после)
- CTA: «Получить консультацию»
- Форма заявок на главной
