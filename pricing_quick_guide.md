# 🎯 Быстрая Справка: Структура Услуг ИИ-компаний

## Универсальная структура тарифов (5-6 уровней)

```
Бесплатно → Бюджетный ($8-15) → Стандарт ($20-30) → Профессионал ($100-200) → Enterprise (договорная)
```

---

## ✅ Типовые пакеты успешных компаний

### **OPENAI (ChatGPT)**
```
Free → Go ($8) → Plus ($20) → Pro ($200) → Business ($25/чел) → Enterprise
```
- 6 уровней
- По пользователю для Business
- Дифференциация по моделям ИИ
- Органический рост: Free → Go → Plus → Pro

### **RUNWAY (Видео ИИ)**
```
Free → Standard ($12) → Pro ($28) → Unlimited ($76) → Enterprise
```
- 5 уровней
- На основе кредитов (625-2250/месяц)
- Дифференциация по функциям
- Безмесячный бюджет: $12 за Standard для малого бизнеса

---

## 📊 Что выделяют в Базовом / Стандарте / Премиум

### **Базовый план (~$8-15)**
- Основные модели/функции
- Ограниченный объем (250-625 единиц)
-限制 по скорости обработки
- Базовая поддержка
- Для индивидуалов и небольших проектов

### **Стандарт (~$20-30)**
- Все основные функции
- Расширенный объем (625-2000 единиц)
- Нормальная скорость
- Поддержка по email
- Для малых команд (2-5 человек)

### **Pro/Премиум (от $50+)**
- Полный функционал
- Неограниченный/максимальный объем
- Максимальная скорость
- Приоритетная поддержка
- Для профессионалов/частых пользователей

### **Business ($25-50/чел)**
- Команда управление
- Admin консоль
- SSO/SAML
- Интеграции (60+)
- Поддержка 24/5
- Для малого бизнеса (5-10 чел)

### **Enterprise (договорная)**
- Кастомные объемы
- Максимальная безопасность (SOC 2, ISO)
- Персональный менеджер
- Интеграции на заказ
- Поддержка 24/7
- Для крупных организаций

## 🎯 6 Главных Категорий Услуг

### 1. **Модели & Рассуждения**
   - Базовые модели
   - Расширенные модели
   - Режимы с глубоким анализом

### 2. **Контент-генерация**
   - Тексты
   - Изображения
   - Видео
   - Аудио/Речь

### 3. **Анализ & Обработка**
   - Документы (OCR)
   - Изображения
   - Видео
   - Данные

### 4. **Автоматизация**
   - Рабочие процессы
   - API доступ
   - Интеграции
   - Кастомные агенты

### 5. **Команда & Управление**
   - Учетные записи пользователей
   - Администрирование
   - Аналитика
   - Контроль доступа

### 6. **Безопасность & Соответствие**
   - Шифрование
   - SSO
   - Соответствие (SOC 2, ISO)
   - Конфиденциальность

## 🔑 Ключевые Метрики Дифференциации

| Метрика | Free → Pro |
|---|---|
| **Объем использования** | 125 → неограниченно |
| **Скорость обработки** | Медленно → максимум |
| **Модели** | Базовые → все |
| **Хранилище** | 5 GB → 500 GB |
| **Пользователей в команде** | 1 → 10+ |
| **Проекты** | 3 → неограниченно |
| **Интеграции** | Никакие → 60+ |
| **Поддержка** | Нет → 24/7 VIP |
| **Безопасность** | Стандартная → максимум + кастом |
| **Функции** | Частичные → все |

- [ ] Выбрать основной ограничитель (объем, скорость, функции?)

- [ ] Добавить скидку 15-20% за годовую оплату
- [ ] Выделить Team/Business план отдельно
- [ ] Зарезервировать Enterprise с договорной ценой
- [ ] Тестировать реакцию на цены
- [ ] Обновлять каждые 6-12 месяцев

## 🌟 Лучшие практики структурирования

1. **Всегда есть Free** (привлечение)
2. **Шаг между планами ~2-3x** по цене
3. **Не более 6 уровней** (запутанность)
4. **Team план** отделен от индивидуальных
5. **Enterprise всегда договорная** (премиум ощущение)
6. **Основной ограничитель один** (либо объем, либо функции)
7. **Скидка за годовую оплату** 15-20%
8. **Честное описание** чем отличается каждый уровень
9. **Месячная гибкость** (легко переходить между планами)
10. **Часто задаваемые вопросы** про переходы и апгрейды
