# Анализ структуры услуг и тарифов компаний в сфере ИИ и автоматизации

## 1. ОСНОВНЫЕ КАТЕГОРИИ УСЛУГ

### OpenAI (ChatGPT, API)
- **Модели ИИ** (GPT-5.3, GPT-5.4, GPT-5 Thinking)
- **Генерация контента** (текст, изображения, видео Sora, аудио)
- **Анализ данных** (обработка файлов, интерпретация данных)
- **Автоматизация** (агент Codex, интеграции, рабочие процессы)
- **Разработка** (API, кастомные GPT, приложения)
- **Корпоративные функции** (управление, безопасность, соответствие)

### Runway (Generative AI для медиа-контента)
- **Генеративное видео** (Gen-4.5, Gen-4, тексту видео и образ в видео)
- **Генеративные изображения** (создание, редактирование, апскейлинг)
- **Генеративное аудио** (синтез речи, текст-в-речь)
- **Видеоредак­тирование** (профессиональные инструменты)
- **Рабочие процессы** (автоматизация творческих процессов)
- **Корпоративное хранилище** (облачное хранение активов)

## 2. СТРУКТУРА ТАРИФНЫХ ПЛАНОВ

### Типовая структура: 5-6 уровней цены

#### **OpenAI (ChatGPT)**
1. **Free** — 0$ (для пользователей)
2. **Go** — 8$/месяц (бюджетный платный)
3. **Plus** — 20$/месяц (для активных пользователей)
4. **Pro** — 200$/месяц (продвинутый)
5. **Business** — 25$/пользователь/месяц (командные рабочие пространства)
6. **Enterprise** — договорная цена (крупные организации)

#### **Runway**
1. **Free** — 0$ (125 кредитов одноразово)
2. **Standard** — 12$/пользователь/месяц (625 кредитов/месяц)
3. **Pro** — 28$/пользователь/месяц (2250 кредитов/месяц)
4. **Unlimited** — 76$/пользователь/месяц (неограниченные генерации в Explore режиме)
5. **Enterprise** — договорная цена (кастомизация, безопасность, поддержка)

## 5. ПАТТЕРНЫ ЦЕНООБРАЗОВАНИЯ

### **Модель 1: По пользователю (Per-Seat)**
*OpenAI Business, Runway Standard/Pro/Unlimited*
- Цена на одного пользователя в месяц
- Шкала скидок за годовую оплату (10-20%)
- Идеально для командной работы

### **Модель 2: На основе кредитов (Credit-Based)**
*Runway*
- Фиксированное количество кредитов в месяц
- Эквиваленты в секунды видео / количество изображений
- Некредитованные кредиты переносятся или исчезают
- Возможность дополнительной покупки

### **Модель 3: На основе доступа компонентов (Feature-Based)**
*OpenAI Free/Plus/Pro*
- Разные модели в разных планах
- Прогрессирующий доступ к функциям
- Различные скоростные режимы

### **Модель 4: Enterprise (договорная)**
*Обе компании*
- Кастомные объемы
- Персональный успех-менеджер
- Интеграции на заказ
- Расширенная безопасность / соответствие
- Приоритетная поддержка 24/7

## 6. КЛЮЧЕВЫЕ РАЗЛИЧИЯ В СТРУКТУРИРОВАНИИ

### OpenAI (Чат-ориентированная)]
✓ Фокус на разных моделях (качество)

✓ Режим исследований (Explore) для неограниченных генераций
✓ Творческие интеграции (Apps, Workflows)

## 9. РЕКОМЕНДАЦИИ ДЛЯ СОБСТВЕННОГО ПРОЕКТА

Если вы создаете сервис ИИ-автоматизации, рассмотрите:

### Для базовой структуры:
1. **Free** (125-625 юнитов) — привлечение пользователей
2. **Starter/Базовый** (~$10-15) — активные индивидуальные пользователи
3. **Pro/Стандарт** (~$30-50) — частые пользователи, малые команды
4. **Business** (~$99-149) — средние команды с управлением
5. **Enterprise** — договорная цена с кастомизацией

### Ключевые дифференциаторы:
- **Скорость ** (ограничения/минуты обработки)
- **Объем** (кредиты, API вызовы, хранилище)
- **Функции** (модули, интеграции, рабочие процессы)
- **Безопасность** (SSO, соответствие, DLP)
- **Поддержка** (уровни по времени отклика)
