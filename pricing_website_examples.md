## Вариант 2: Для творческих инструментов (как Runway)

```
┌────────────────────────────────────────────────────────┐
│        ВЫБЕРИТЕ СВОЙ ПЛАН ГЕНЕРАЦИИ КОНТЕНТА         │
└────────────────────────────────────────────────────────┘

              Поэкспериментируй (Бесплатно)
                        ↓
             Работай регулярно ($12/мес)
                        ↓
             Профессионально ($28/мес)
                        ↓
         Неограниченный режим создания ($76/мес)
                        ↓
        Корпоративное решение (договорная цена)

┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│     БАЗОВЫЙ    │  │     ПРО        │  │   НЕОГРАНИЧЕНО │
│   Бесплатно    │  │   $28/месяц    │  │    $76/месяц   │
│   125 кредитов │  │  2250 кредитов │  │  неограничено  │
└────────────────┘  └────────────────┘  └────────────────┘

КРЕДИТЫ В ВИДЕО (примерно):
• 125 кредитов = 5 сек видео Gen-4 или 12 изображений Gen-4
• 625 кредитов = 25 сек видео Gen-4 или 62 изображения
• 2250 кредитов = 90 сек видео Gen-4 или 225 изображений

┌────────────────────────────────────────────────────────┐
│        ОПРЕДЕЛИ, ЧТО ТЕБЕ НУЖНО                       │
├────────────────────────────────────────────────────────┤
│ Тип работы        │ Нужный план                        │
├──────────────────┼────────────────────────────────────┤
│ Тестирование     │ Базовый (125 кредитов)             │
│ Частое творчество│ Стандарт ($12, 625 кредитов)       │
│ Профессиональная │ Про ($28, 2250 кредитов)          │
│ работа           │                                    │
│ Режиссура/Студия │ Неограничено ($76)                │
│ Компания/Agency  │ Enterprise (договорная)            │
└──────────────────┴────────────────────────────────────┘
```

## Структурирование категорий на сайте

### Укажи эти разделы:

```
1️⃣ ФУНКЦИИ
   ├─ По категориям (что можешь делать)
   ├─ Сравнение планов (таблица)
   ├─ Калькулятор стоимости (если сложно считать)
   └─ Демо каждой функции

2️⃣ ТАРИФЫ
   ├─ Основные 4-6 планов (как в вариантах выше)
   ├─ Переключатель Month / Year (с процентом скидки)
   ├─ Кнопки выбора (Get Started / Learn More)
   └─ FAQ про переходы между планами

3️⃣ ДЛЯ БИЗНЕСА
   ├─ Отдельный план Team/Business
   ├─ Преимущества управления
   ├─ Безопасность и соответствие
   ├─ Демо для менеджеров
   └─ Контакты отдела продаж

4️⃣ ENTERPRISE
   ├─ Что включено
   ├─ Примеры клиентов
   ├─ Процесс подбора
   └─ Форма "Связаться с продажами"

5️⃣ FAQ
   ├─ Как выбрать план?
   ├─ Что входит в каждый план?
   ├─ Как изменить или отменить подписку?
   ├─ Какиеметоды оплаты?
   └─ Будут ли новые функции?
```

## 🎯 Лучших практик:

✅ **Делай цены видными сразу** (наверху)
✅ **Один основной план** (most popular)
✅ **Честно пиши, что включено**
✅ **Покажи переводы между планами**
✅ **Предложи пробный период для Premium**
✅ **Год дешевле, чем месяц на 15-20%**
✅ **Бесплатный план всегда есть** (привлечение)
✅ **FAQ про升级/downgrade**
✅ **Live chat для вопросов о выборе**
✅ **Используй зелёный цвет для CTA** (кнопки)
