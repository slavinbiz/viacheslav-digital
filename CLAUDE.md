# Правила работы с проектом

Привет! Здесь краткие и понятные правила, чтобы все работало просто и дружелюбно.

1. Все страницы делаем в одном HTML-файле
   - Один HTML-файл на проект. Без создания множества html-страниц.
   - Без внешних библиотек (jQuery, Bootstrap, React и т.п.). Только чистый HTML/CSS/JavaScript, если нужен JS.

2. Дизайн
   - Тёмный фон: основной фон для страницы тёмный.
   - Светлый текст: контент читаемый на тёмном фоне, светлые заголовки и текст.
   - Акцентный цвет: оранжевый (используется для кнопок, ссылок, ключевых выделений).

3. Адаптивная вёрстка
   - Страница должна корректно отображаться на мобильных и десктопных экранах.
   - Используем относительные размеры, медиа-запросы, флекс/грид.
   - Текст, кнопки, блоки должны масштабироваться под экран.

4. Стиль текста
   - Простым и дружелюбным языком.
   - Без канцелярита, без сухих формулировок.
   - Короткие понятные фразы, чтобы любой мог читать легко.

---

До начала создания финального HTML-файла сначала согласуем концепцию и основные секции. Как только будем уверены, переходи к реализации.
